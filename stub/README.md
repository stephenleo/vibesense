# vibesense → @vibesense/cli

This package has moved to [`@vibesense/cli`](https://www.npmjs.com/package/@vibesense/cli).

```sh
npm install -g @vibesense/cli   # or: npx @vibesense/cli
```

The `vibesense` name is kept only as a redirect so old install instructions fail with a pointer instead of a 404. It is published manually (`cd stub && npm publish`), never by CI.
